# Linows 7 Editions

Linows 7 has 4 editions to be compatible with different taste.

<br><br>

# **Linows 7 Starter – Features**
The lightweight edition focused on performance.

## Visual
- Basic Windows-7-inspired theme  
- No blur  
- Default KDE animations  
- Win7-style icon theme  
- Classic cursor set  
- Minimal taskbar and menu layout  

## Sound & Fonts
- Minimal sounds (no startup and logout sounds)
- No boot splash screen
- Just Segoe UI font for compatibility

## Content
- 3 default wallpapers (2 for desktop and 1 for lock screen)
- Optional desktop shortcuts (Computer, Home, Trash)

<br><br>

# **Linows 7 Home – Features**
A complete aesthetic experience without deep application theming.

## Visual
- Full Aero theme  
- Light transparency  
- Soft blur  
- KDE animations enabled  
- Full Win7-style icon set  
- Win7-inspired cursors  
- Taskbar configured to match Windows 7 behavior  

## Sound & Fonts
- Windows-7-style sound theme  
- Classic splash screen  
- Segoe UI font included  

## Content
- Windows 7 wallpaper collection  
- Optional desktop shortcuts (Computer, Home, Trash)

<br><br>

# **Linows 7 Premium – Features**
Adds Aero Glass effects and themed applications.

## Visual
- Everything from Home  
- Stronger Aero Glass effect  
- Increased blur radius  
- Windows 7 widgets  
- Enhanced icons and cursors  

## Sound & Fonts
- Full custom sound theme  
- Login and logout sounds  
- Segoe UI and Lucida Console fonts  

## Themed Applications
- Firefox with Aero userChrome  
- Terminal with translucent theme  
- Notepad-style text editor  
- System Monitor styled like Task Manager  
- Calculator styled after Win7  
- Paint-like drawing app  
- LibreOffice with Win7 appearance  

## Content
- Extra wallpaper pack  
- Classic-style screensavers  
- Mini utilities  
- Demo photos, videos, and music  

<br><br>

# **Linows 7 Ultimate**
The full system transformation edition.

## Visual
- Everything from Premium  
- Complete Aero Glass across the system  
- Strong blur, glow, and shadows  
- Full SDDM login theme  
- Plymouth boot animation  
- Custom drive and folder icons  
- GTK theme for non-KDE apps  
- Taskbar configured with Win7 grouping and previews  

## Sound & Fonts
- Complete sound theme  
- Startup and shutdown sounds  
- Segoe UI, Lucida Console, TerminalEvctor  

## Themed Applications
- Firefox with Aero Glass  
- Aero-styled Terminal  
- Notepad style  
- Task Manager style  
- Calculator, Paint, Media Player  
- LibreOffice with Win7 look  
- Included games: Space Cadet, Solitaire, Minesweeper, etc.  

## Content
- Full wallpaper collection  
- Demo photos, music, and videos  
- Profile icons  
- Classic widgets  
- Desktop shortcuts (Computer, Home, Trash)  
- Optional “Old Google” and “Old YouTube” links  

## System-Level (Optional)
- **Hostname set to “linows7” (optional during installation)**  
- **OS name changed to “Linows 7 Ultimate” (optional)**  

These options can be skipped without affecting the rest of the theme.
